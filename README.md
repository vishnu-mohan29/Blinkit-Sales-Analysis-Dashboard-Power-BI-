#  Blinkit Sales Analysis Dashboard (Power BI)

##  Overview
This project presents an interactive Power BI dashboard analyzing Blinkit sales data. The goal is to extract meaningful insights related to sales performance, product categories, and outlet characteristics.

---

##  Objectives
- Analyze total and average sales performance
- Identify top-performing product categories
- Evaluate outlet performance by size and location
- Understand customer preferences based on product attributes

---

##  Key Metrics
- 💰 Total Sales
- 📦 Number of Items
- 📈 Average Sales
- ⭐ Average Rating

---

##  Dashboard Insights

### 1. Sales Performance
- Strong overall revenue of $1.2M
- Consistent sales trend with slight fluctuations over time

### 2. Product Analysis
- Top categories: Fruits, Snacks, Household Items
- Lower-performing categories: Seafood, Breakfast items

### 3. Outlet Analysis
- Tier 3 locations generate the highest sales
- Medium-sized outlets contribute the most revenue

### 4. Customer Preferences
- Regular fat products dominate sales compared to low-fat items
- Average ratings remain consistent across outlet types
